# ubuntu-dotfiles
Different types of configuration files for setting up an Ubuntu.
Includes:
- Fonts
- Background images
- .bashrc aliases
- .zshhrc complete configurations (themes, plugins and more)
Should work with most Debian based linux distros.

## Requirements
### Terminal
- ZSH shell
- OhMyZsh Framwork
- Powerlevel 10k theme
- Curl
- Stow
